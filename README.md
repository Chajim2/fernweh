Fernweh is a journaling app that analyzes your writing and helps you identify your emotions.
Newly added vector database search that can help you connect with friends who feel the same as you.
Download at: https://chajim2.itch.io/fernweh

**Technologies**: Python - Flask, SQLite, sqlite-vector, Kivy (Legacy, replaced by Flutter)
